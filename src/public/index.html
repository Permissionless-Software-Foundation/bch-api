BCHA API

The REST API for the Cash ABC blockchain.


bch-api is a REST API, allowing modern applications to interact with the BCHA blockchain. New to the BCHA blockchain? Check out these bitcoinabc.org.


A REST API can work with any programming language, but bcha-api is intended to be paired with the npm bch-js JavaScript library. Here is example code for common BCHA use cases. slp-cli-wallet is another npm library that provides common wallet functionality, to get your apps up and running as quickly as possible.


While you'll be able to run through the examples without one, you should sign-up for a free API key in order to unlock higher rate limits.
